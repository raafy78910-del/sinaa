# نظام شؤون الموظفين المدرسي
نظام ويب مجاني بالكامل لإدارة شؤون موظفي المدارس، مبني على **Google Sheets** كقاعدة بيانات.

## الملفات
| الملف | الوصف |
|-------|-------|
| `index.html` | واجهة النظام الكاملة — افتحها مباشرة بالمتصفح أو استضفها على GitHub Pages |
| `Code.gs` | كود الخادم — يُنسخ كاملاً داخل Google Apps Script |
| `SETUP_GUIDE.md` | دليل الإعداد خطوة بخطوة |

## الوحدات
- **بيانات الموظفين** — إضافة وتعديل وحذف
- **التشكيلات المدرسية** — إدخال حصص كل معلم لكل شعبة + تفريغ
- **جدول التشكيلات** — مصفوفة معلم × شعبة مع تصدير Excel مُنسَّق (ألوان + تجميد + RTL) وطباعة PDF
- **إجازات الموظفين**
- **طلبات المواصلات** — نموذج بدل مواصلات مطابق للنموذج الرسمي لوزارة التربية والتعليم الفلسطينية
- **النقل والتقاعد والاستقالة**
- **التكليفات والمخالفات**
- **التقارير** — تصدير Excel / طباعة PDF لكل وحدة
- **إدارة المستخدمين** — صلاحيات (مدير / سكرتير / اطّلاع)

## الإعداد السريع
1. افتح [sheets.google.com](https://sheets.google.com) وأنشئ ملفاً جديداً
2. الإضافات → Apps Script → الصق محتوى `Code.gs` → شغّل `setupSheets`
3. نشر → New deployment → Web app → Who has access: Anyone → انسخ الرابط
4. افتح `index.html` وأدخل الرابط في شاشة الربط

> راجع `SETUP_GUIDE.md` للتفاصيل الكاملة

## الاستضافة على GitHub Pages (اختياري)
1. ارفع الملفات لمستودع GitHub
2. Settings → Pages → Branch: main → Save
3. سيظهر رابط ثابت شكله: `https://username.github.io/repo-name`

## الأمان
- كلمات المرور مشفرة SHA-256 قبل إرسالها (لا تُخزَّن كنص عادي)
- التحقق من الصلاحيات يتم في الخادم (Code.gs) وليس فقط في الواجهة
